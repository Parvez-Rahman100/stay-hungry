Local Storage Vs Session Storage

Local Storage

Local storage is a memory that keeps data from website, it doesn't remove data automatically.Local storage will be stored untill you delete data or users delete data.It is used for long term.It can store more data than session storage.

Session Storage

Session storage is a memory that keeps data from website, as long as browser's tab is open.When tab will be closed,data that was stored in session storage will be erased.It is used for temporary.It can store less data than local storage.
